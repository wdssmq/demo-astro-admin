## 命令

所有命令请在项目根目录的终端中运行：

| Command        | Action                                  |
| :------------- | :-------------------------------------- |
| `pnpm install` | 安装依赖                                |
| `pnpm dev`     | 在 `localhost:4321` 启动本地开发服务器  |
| `pnpm build`   | 将构建输出到 `./dist/` ，可用于静态展示 |
